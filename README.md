
Project Name:
--Keen Keeper Apps

A short description:

--Keen Keeper is a smart personal management web application designed to help users track interactions, manage relationships, and stay organized efficiently. It allows users to log calls, messages, and activities in a timeline format, making it easy to monitor communication history and maintain consistency in follow-ups.

Technologies I Used:

--- Next.js
--- React.js
--- Tailwind CSS
--- Context API (State Management)
--- Recharts (for data visualization)
--- Vercel (Deployment)
